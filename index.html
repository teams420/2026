Setup Manual & Guide

Welcome! This panel is designed to manage your dynamic Android Prompt app layout. It utilizes Firebase Firestore (for live synchronization) and Cloudflare R2 Bucket storage API (AWS S3 SDK) for media files.

1. CORS Policy Setup

To let your administration panel upload files to your bucket dynamically, configure the CORS property inside your Cloudflare R2 bucket settings:

[
  {
    "AllowedHeaders": ["*"],
    "AllowedMethods": ["GET", "PUT", "POST", "DELETE", "HEAD"],
    "AllowedOrigins": ["*"],
    "ExposeHeaders": []
  }
]
2. Sandbox Mode Fallback

If you proceed without credentials, the system automatically falls back to Sandbox Mode. All categories, prompts, branding assets, and file uploads (saved instantly as cached Base64 local URLs!) will operate directly in your browser's persistent cache. Once you plug real keys, the system hooks up immediately!
