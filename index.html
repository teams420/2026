Welcome! This admin panel connects securely and directly to your **Firebase Firestore** and **Cloudflare R2 Storage Bucket** to allow dynamic updates parameters automatically without code rebuilds.

Connecting R2 Bucket:

Add CORS configuration to your Cloudflare R2 bucket settings in order to permit dynamic upload operations from the web browser dashboard.

Offline Sandbox Mode:

If no keys are entered in settings, the panel operates in local sandbox mode using cache to let you preview layouts, save categories, and manage prompt ideas immediately.
