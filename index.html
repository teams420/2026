Setup configuration Guide

Welcome! This SPA is designed as a direct bridge to manage your Android Prompt App. It utilizes Firebase Firestore (for real-time database syncing) and Cloudflare R2 Bucket storage via AWS S3 compatibility.

1. Setting up Cloudflare R2 (S3 Compatible API)

Ensure you configure CORS on your Cloudflare R2 bucket to accept uploads from endpoints or local servers (e.g., allow your GitHub Pages origin URL):

[
  {
    "AllowedHeaders": ["*"],
    "AllowedMethods": ["GET", "PUT", "POST", "DELETE", "HEAD"],
    "AllowedOrigins": ["*"],
    "ExposeHeaders": []
  }
]
2. Edit Credentials Block

To establish dynamic connections, open this file in any text editor and fill in the credentials inside the firebaseConfig and r2Credentials constants declared at the top of the JavaScript code.
